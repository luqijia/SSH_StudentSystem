# SSH_StudentSystem
Spring+Struct2+Hibernate+前端框架Bootstarp+Jquery+h-ui+Layui+Datatables实现简单的学生管理系统，目的主要是使用Hibernate进行CRUD操作

数据库设计参见ssh_studentsystem.sql
